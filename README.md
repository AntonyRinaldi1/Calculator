# Calculator Web Application

A simple and responsive Calculator Web Application built using HTML, CSS, and JavaScript. This calculator performs basic arithmetic operations such as addition, subtraction, multiplication, and division through a user-friendly interface.

## Features

- Simple and attractive user interface
- Responsive design
- Addition (+)
- Subtraction (-)
- Multiplication (×)
- Division (÷)
- Decimal number support
- Clear display functionality
- Error handling for invalid expressions

## Technologies Used

- HTML5
- CSS3
- JavaScript
